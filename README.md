# ??? AI Gesture Mouse Control

Control your computer mouse using hand gestures via your webcam. This project uses **MediaPipe** for hand landmark detection and **PyAutoGUI** for seamless mouse interaction.

## ?? Gestures
* **Move Mouse:** Index finger alone
* **Left Click:** Thumb + Index pinch
* **Right Click:** Thumb + Middle pinch
* **Scroll:** Index + Middle extended (move hand up/down)
